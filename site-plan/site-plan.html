Overview

Purpose

Create an agent creator that allows the user to create an agent for VALORANT, a video game based on characters with pre-designated abilities.

Audience

VALORANT players who want to create an agent but haven't had a tool built for that specific purpose.

Dynamic elements

JavaScript will be used to display an unknown amount of agents on the home screen, handle which role or ability is active and replaces navigation button functionality based on current navigation paths.

Content

Home page

The logo will be in the top left corner that navigates back to the home page. Agent spotlights, showing the agent name and image. Clicking it will navigate to the view page with that agents information.

Create page

The logo will be in the top left corner that navigates back to the home page. Fields will allow input for text and images.

View page

The logo will be in the top left corner that navigates back to the home page. The agent data will be displayed without the capability to edit.
